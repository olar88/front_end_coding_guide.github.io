命名

基本命名方式
  • 駝峰式命名（Camel case）
    • 就像駝峰會一高一低
    • 單字之間不空格，連起來的第一個字大寫
    • 例如：有一個名為 handleAddInput 的函數， 小駝峰為 : handleAddInput、大駝峰為 : HandleAddInput。

  • 蛇型命名（Snake case）
    • 利用底線來連接單字，全部都是小寫
    • 同上述範例，蛇型命名為：handle_add_input

  • 烤肉串命名（Kebab case）
    • - 符號將自串在一起，全部都是小寫
    • 同上述範例，會變成：handle-add-input

各種場合需使用變數命名方式
  • 布林值

    使用 is + 狀態（動詞過去完成式／現在進行式／形容詞） 表示該狀態是否已達到。

    Ex : isLoaded、isVisible、isChecked、isClick

  • 屬性狀態 (HTML Element or component )

    定義component props name

    name意義
    clickEvent點擊動作
    onMouseEnter滑鼠移入動作
    disabled是否禁用
    checked是否選中
    text渲染文字
    value渲染值

  • 變數 (狀態) 值
    • 選單：codeList_選單名稱 (大駝峰式)

      Ex : codeList_SEX 、codeList_Visible 、codeList_CareerRank

    • 頁面 state：請使用小駝峰命名
      基本 state name意義
      pageSearch 頁面搜尋條件
      pageAdd 欲新增資料
      pageUpdate欲更新資料
      pageInfo編輯頁查詢出資料(不會更動)
      API 回傳資料意義
      resDArrAPI 回傳資料為陣列
      resDataAPI 回傳資料為物件或單一值
    • 處理資料：請使用小駝峰命名
      基本 資料 name意義
      item物件
      element / elem / ele物件
      index / indx / idx索引值
      seq索引值
      event / e事件

      透過處理取得之資料請以 「動詞+物件+條件」 組合命名

      Ex : getItemById、foundIndexByID

      function formatData(){
		// 第一層 loop
		for(let groupIndex in group){
				let groupItem = group[groupIndex];

				// 第二層 loop
				groupItem.child.map((childItem ,childIndex)=>{
						//... code ... //
				})
		}
}

    • 錯誤範例：請避免以下類型命名
      錯誤範例
      nMeaningless
      nErrAmbiguous abbreviation.
      wgcConnectionsOnly you knows what this stands for
      pcReaderLots of things can be abbreviated "pc"
      kSecondsPerDayDo not use Hungarian notation

      不要使用項目以外的讀者不明確或不熟悉的縮寫，也不要透過刪除單字中的字母來縮寫。

  • 元件 Element

    class component、function component、interface、typedef、record

    都請用大駝峰法命名

    Ex : OutlineInput、ModalAddInner、PageProps

  • Method / Function

    請用小駝峰法命名, 並清楚以該功能項為名稱

    Ex :

    輸入搜尋條件: searchHandleInput 、

    送出訊息: sendMessage、

    重構資料: formatData、

  • Enum 列舉

    請用大駝峰法命名列舉名稱，列舉職請用全大寫註釋

    Ex : 

    enum TabName　= {
  ORDER: 'order',
  ACCOUNT: 'account',
	CAR_INFO: 'carInfo',
};
